# 🫀 GUI para Predicción de Enfermedades Cardíacas 💻

Esta aplicación de escritorio, desarrollada en **Python** con **Tkinter**, permite predecir enfermedades cardíacas utilizando un modelo de **K-Nearest Neighbors (KNN)** optimizado con **GridSearchCV**. La interfaz es moderna, intuitiva y elegante, gracias a `sv-ttk`.

## 🚀 Características
✔ **Interfaz moderna y estilizada** con una distribución optimizada de inputs.  
✔ **Placeholders dinámicos** con información detallada sobre cada campo.  
✔ **Modelo de predicción basado en KNN**, con datos preprocesados y escalados.  
✔ **Limpieza automática de inputs** después de cada predicción.  
✔ **Mensajes informativos** para guiar al usuario en la introducción de datos.  
✔ **Framework escalable**, adaptable para mejoras futuras y otros modelos de Machine Learning.

## 🛠️ Tecnologías utilizadas
- **Python 3**
- **Tkinter** (Interfaz gráfica)
- **sv-ttk** (Estilo moderno)
- **Pandas** (Gestión de datos)
- **NumPy**
- **scikit-learn** (Machine Learning)

## 🎯 Uso
1. **Ejecuta el script** en Python.
2. **Ingresa los datos del paciente** en los campos correspondientes.
3. **Haz clic en "Predecir"** para obtener el resultado basado en el modelo entrenado.
4. **El sistema limpiará los datos automáticamente** para facilitar nuevas predicciones.
